Справочник "Виды ремонтов"

Перечень ремонтов и параметры для планирования и учета расходов, связанных с выполнения ремонтных работ, хранятся в справочнике Виды ремонтов.

Справочник Виды ремонтов доступен только при подключении модуля Управление транспортом и может быть открыт при помощи одноименной команды панели навигации раздела Сервис и настройки из группы ГСМ и ремонты.

Для каждого вида ремонта указывается:

  • Наименование понятное пользователям системы.

  • Тип ремонта, который определяет характер ремонта и влияет на доступность реквизитов в документе Ремонт (ТО), а также может использоваться в аналитической отчетности для отборов и группировок.

  • Статья расходов для регистрации расходов, связанных с выполнением ремонтных работ.

  • Плановое время выполнения работ в часах и минутах, которое используется для планирования ремонтов. 

  • На закладке Работы (по умолчанию) указывается перечень работ и их статьи расходов. Данные работы по умолчанию отображаются на закладке Работы в документе Ремонт (ТО) при выборе вида ремонтных работ. 

  • На закладке Комментарий указывается дополнительная информация по указанному Виду ремонта.
