Справочник "Экземпляры документов"

Для каждого экземпляра документа заполняется Наименование, Номер экземпляра и Рейс, в котором был сформирован данный документ. 

Номер рейса устанавливается в справочнике Экземпляры документов при переводе Рейса в статус К выполнению, где по Виду перевозки определяется способ учета возвращенных документов и формируются движения в регистр Непереданные документы.

  • При выборе С точностью до типов документов, учет ведется по типам документов.
  • Если выбран С точностью до экземпляра документов, то определяется пакет печатных форм, которые необходимо вернуть.
Для Рейса он задается в реквизите Пакет печатных форм по рейсу перевозчика, для Задания на перевозку груза в реквизите Пакет печатных форм по заданию получателя задания.
