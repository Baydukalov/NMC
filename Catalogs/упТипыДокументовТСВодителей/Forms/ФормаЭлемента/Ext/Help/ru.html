Справочник "Типы документов транспортных средств / водителей"

Справочник Типы документов транспортных средств/водителей может быть открыт при помощи одноименной команды панели навигации раздела Управление транспортом из группы Управление документами документами.

Для каждого типа документа указывается:

  • В реквизите Документ выдается для какого объекта учета выдается этот тип документа. Документ может быть оформлен на водителя, транспортное средство или одновременно на водителя и транспортное средство.

    Примером такого документа может быть страховка.

  • Опционально на какое количество транспортных средств и/или водителей может быть выдан этот тип документа, а именно на одного, нескольких или без ограничений.

  • В группе реквизитов Срок действия каким образом определяется срок действия документов этого типа. Опциально может быть выбран один из трех вариантов:

    • срок действия может быть указан непосредственно в документе этого типа;

    • определяется по сроку действия с даты оформления, при этом срок действия указывается для типа документов, а дата оформления в документе;

    • неограниченный срок действия.

  • Использование механизма уведомления при наступлении окончания срока действия документа, которое регулируется флагом Предупреждать об окончании срока действия документа. Если информацию об окончании срока действия документа необходимо получать заранее, то количество дней за которое необходимо выводить предупреждение указывается в реквизите Предупреждать за:.

Для контроля сроков действия документов при исполнении рейсов типы документы могут использоваться при формировании пакетов документов для контроля.
