Справочник "Показатели расчета тарифов"

Показатели, используемые при формировании правил расчета тарифов, описываются в справочнике Показатели расчета тарифов. Справочник Показатели расчета тарифов доступен только при подключении модуля Управление перевозками, Управление транспортом и может быть открыт при помощи одноименной команды панели навигации раздела Сервис и настройки из группы Тарификация.

Для каждого показателя указывается тип показателя и параметр расчета. Значение реквизита Тип показателя может принимать одно из двух значений: расчетный или фиксированный.

Если выбран тип показателя расчетный, то в этом случае в реквизите Параметр расчета необходимо из предопределенного списка параметров выбрать одни из доступных параметров. В этом случае алгоритм расчета показателя определяется правилами определенными разработчиками системы.

Для некоторых параметров в табличной части формы элемента показателя возможно уточнение размера показателя в зависимости от значения параметра.

Если выбран тип показателя фиксированный, то значение показателя устанавливается при помощи документа Установка значений фиксированных показателей.
