Справочник "Цветовые шкалы"

Цветовые шкалы используются для отображения в РМ: Спутниковый мониторинг на электронной географической карте трека в цветах, значение цветовой шкалы соответствует скорости движения транспортного средства. Перечень цветовых шкал хранится в одноименном справочнике.

Для каждой шкалы указывается:

  • Наименование понятное пользователям системы.
  • Толщина линии, которая будет использоваться при отображении трека.
  • Прозрачность, которая будет использоваться при отображении трека.
  • Шкала интервалов, которая будет использоваться при отображении трека на электронной географической карте.
