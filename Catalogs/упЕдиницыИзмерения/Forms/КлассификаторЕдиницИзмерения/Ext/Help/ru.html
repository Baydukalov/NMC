Справочник "Единицы измерения"

Единицы измерения, которые могут использоваться при описании номенклатурного состава груза хранятся в справочнике Единицы измерения.

Справочник Единицы измерения может быть открыт при помощи одноименной команды панели навигации раздела Нормативно-справочная информация из группы Номенклатура.

Перечень единиц изменения может быть создан с при помощи подбора из Общероссийского классификатора единиц измерения.
