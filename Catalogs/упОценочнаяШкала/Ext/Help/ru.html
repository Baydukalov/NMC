Справочник "Оценочная шкала"

Элементы справочника Оценочная шкала используются при формировании правил расчета тарифов. 

Для каждой шкалы указывается наименование, идентификатор, используемый в формулах и непосредственно шкала.

Шкала описывается перечнем интервалов. Для каждого интервала шкалы указывается начальное и конечное значение оценки интервала и значение показателя для этого интервала. Представление значения формируется автоматически в колонке Представление.
