Справочник Типы контейнеров

Справочник Типы контейнеров используется для описания единых классификационных признаков контейнеров, определяющих его назначение и общее конструктивное исполнение. В зависимости от параметров типа контейнеров в системе осуществляется подбор заданий в операции с контейнерами при этом сравниваются значение массы и объема, заданные для типа контейнеров с суммарной массой и объемом включенных заданий на перевозку.

Для каждого типа транспортных средства указываются основные характеристики:
  • Наименование – указывается наименование понятное пользователю.
  • Весогабаритные характеристики в реквизитах Ш х Д х В, Грузоподъемность и Объем, Количество мест, Количество загрузочных метров, Масса контейнера.
  • Тип груза контейнера, при формировании Заявки на перевозку груза на основании Операции с контейнером указанный тип по умолчанию устанавливается в тип груза в заявке.
  • Температурный режим, который обеспечивается при перевозке в контейнерах этого типа.

На закладке Классы опасности отдельно указывается перечень классов опасности грузов, которые могут перевозиться в контейнерах данного типа.
