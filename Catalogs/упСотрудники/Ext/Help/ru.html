Справочник "Сотрудники"

В справочнике Cотрудники хранится перечень всех физических лиц, которые могут принимать участие в процессах перевозки грузов.

Для каждого сотрудника указывается:

  • ФИО - для идентификации физического лица.
  • Пол - выбор доступен из списка: Мужской или Женский.
  • Тип сотрудника, который используется для отборов при выборе сотрудника в документах системы. Выбор доступен из списка: Водитель, Водитель-экспедитор, Врач, Экспедитор, Курьер, Логист, Механик, прочее.
  • Партнер, к которому относится этот сотрудник и Подразделение, в котором он числится. Параметры также используются для отбора сотрудников при выборе в документах системы.
  • Если у Сотрудника установлен признак Сотрудник числится в несколькиз организациях, его можно назначать на рейс в форме документа без привязки к Партнеру.
  • Подразделение и Направление деятельности, к которым относится данный сотрудник.
  • Табельный номер, который используется при формировании печатных форм из документов системы.
  • График работы – устанавливается график работы сотрудника.
  • Пользователь. Устанавливается пользователь системы, который имеет права подписи документов.

На закладке Адреса, телефоны указывается контактная информация этого сотрудника.

На закладке Параметры авторизации указываются параметры для авторизации сотрудника в системе в специализированном рабочем месте при въезде и выезде с территории предприятия, а также при исполнении рейса с помощью приложения Android-клиент.
