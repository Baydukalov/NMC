Справочник "Правила расчета тарифов"

Для расчета доходов и расходов предприятия от оказания услуг перевозки грузов для каждой статьи доходов и расходов могут быть указаны правила расчета финансовых показателей. Правила расчета финансовых показателей хранятся в справочнике Правила расчета тарифов. Правила используются в документе Установка правил расчета доходов и расходов. Документ Установка правил расчета доходов и расходов позволяет зафиксировать в системе порядок расчета доходов и расходов по конкретной статье доходов и расходов.

Справочник Правила расчета тарифов доступен только при подключении модуля Управление перевозками и может быть открыт при помощи одноименной команды панели навигации раздела Сервис и настройки из группы Тарификация.

В правилах указывается наименование и устанавливается Способ расчета:

  • Формула.
  • Произвольный.

Формула расчета описывается с использованием синтаксиса языка 1С:Предприятие, при этом могут использоваться условия, функции, показатели, шкалы и тарифные сетки.

Произвольный способ расчёта позволяет использовать файл внешней обработки. Произвольный способ выполняет расчет через внешнюю обработку с возможностью отладки. Для отладки используется Каталог внешних обработок для отладки расположенный в Администрирование, Настройки системы, Сервис и дополнительные настройки. Произвольный способ расчета предназначен для программистов 1С. Алгоритм расчета задается во внешней обработке в предопределенной процедуре Рассчитать (Показатели, ПроизвольныеДанные), в параметры которой передается:

  • Показатели - структура значений показателей
  • Произвольные Данные - произвольные постоянные данные правила, хранимые в элементе справочника<

Показатели, доступные для использования в правиле, указываются в табличной части на закладке Показатели. Показатели могут быть созданы пользователем самостоятельно.

На закладке Функции приведены все функции, которые могут использоваться в правиле. Перечень функций ограничен средствами языка 1С:Предприятие.

На закладке Шкалы приведены варианты шкал, которые могут использоваться в правиле. Варианты шкал создаются пользователями самостоятельно.

Для каждого правила расчета тарифа определяется область действия, которая регулируется при помощи флага Расчет проводится по каждой строке. Если флаг установлен, то формула будет применяться для показателей рассчитанных для каждого грузового места, указанного в заявке на перевозку, задании на перевозку или при отражении фактических расходов или доходов. Итоговая сумма доходов или расходов будет получена суммированием значений, рассчитанных по каждой строке. В противном случае формула будет применяться к общим показателям по документу.

На закладке Тарифные сетки приведены варианты тарифных сеток, которые могут использоваться для в формулах. Тарифные сетки могут быть созданы пользователем самостоятельно.

В правилах расчета тарифов возможно использовать двумерную тарифную сетку в формулах.
