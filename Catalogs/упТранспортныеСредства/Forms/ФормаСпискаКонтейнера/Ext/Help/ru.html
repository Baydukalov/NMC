Контейнеры


Для обозначения в системе контейнеров, предусмотрен справочник Контейнеры. В данном справочнике должны быть заведены все контейнеры, с которыми работает компания, в том числе собственные, наемные или арендуемые.

Для удобства работы пользователей при выполнении процедур поиска и выбора контейнеров в форме списка справочника может быть использован отбор по перевозчику-владельцу контейнеров.

Для ведения списка контейнеров, которые используются при выполнении перевозки, предусмотрен справочник Контейнеры, доступный в панели навигации раздела Транспорт из группы НСИ.

Ведение и организация справочника выполняется на усмотрение Пользователя системы. Контейнеры могут быть сгруппированы, например, по Перевозчикам или по Типам контейнеров. Для этого необходимо создать отдельную группу.

  • Для каждого контейнера указывается Наименование, Номер контейнера.
  • Для контейнера задается принадлежность к конкретному Перевозчику, путем выбора из справочника Партнеров. В дальнейшем используется при подборе ТС перевозчиков при формировании операций с контейнерами.
  • Собственный контейнер – указывается признак для собственного контейнера.
  • Тип контейнера – один из основных классификационных признаков контейнера, определяющий его назначение и общее конструктивное исполнение.
  • Для контейнера может быть задан График работы, который определяет доступность конкретного контейнера в определенный день или временной интервал

На закладке Основных характеристик для контейнеров также задаются внешние габариты, на основании которых производится расчет вместимости контейнеров. Данные по ВГХ могут наследоваться из установленного Типа контейнера.

На основании Объема, Грузоподъемности или Количества грузовых мест, Количества загрузочных метров ведется учет вместимости грузов по Заданиям на перевозку в ходе планирования операций с контейнерами.

При задании Количества загрузочных метров можно рассчитать грузовое пространство, занимаемое грузом в контейнере.

Значение Температурного режима может наследоваться из заданного Типа контейнера. Задание Температурного режима необходимо в случае установки холодильного агрегата в контейнер 
 с целью поддержания определенного температурного режима.

На закладке Классы опасности могут быть перечислены классы опасности грузов, транспортировку которых может осуществлять конкретный контейнер. Если классы опасности грузов не уточняются, то предполагается что данный контейнер может перевозить грузы любого класса опасности.

На закладке Изображение может храниться изображение контейнера.

На закладке Дополнительно указываются также следующие параметры контейнера:

  • Направление деятельности - указывается направление деятельности Организации. По команде Изменить возможно изменить направление деятельности контейнера. Установленное направление деятельности в справочнике контейнеров используется для анализа и распределения доходов и расходов предприятия.
  • Подразделение, указывается подразделение компании, которому принадлежит контейнер.

На закладке Комментарий указывается дополнительная текстовая информация, связанная с эксплуатацией контейнера.
