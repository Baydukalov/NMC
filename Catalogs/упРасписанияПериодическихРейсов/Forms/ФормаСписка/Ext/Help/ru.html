Справочник "Расписание периодических рейсов"

Для создания периодического рейса необходимо заполнить справочник Расписание периодических рейсов.

Справочник Расписания периодических рейсов может быть открыт при помощи одноименной команды панели навигации раздела Планирование рейсов из группы Формирование рейсов. Расписание периодических рейсов задается для рейсов, которые исполняются по установленному периоду.

Для Расписания периодических рейсов задается следующая информация:
  • Наименование – указывается наименование периодического рейса;
  • Период действия – период, в течение которого необходимо формировать рейсы;
  • Признак Активности расписания – при отключенном признаке расписание становится не активным;
  • Транспорт:
    • Перевозчик, Соглашение – устанавливается конкретный партнер-перевозчик и соглашение с перевозчиком, который планируется на исполнение рейса;
    • Тип транспортного средства – устанавливается тип ТС, используемый в рейсе, на основании грузоподъемности которого подбираются задания на перевозку груза;
    • Транспортное средство – устанавливается транспортное средство, которое будет исполнять рейс в случае, если ТС заранее известно;
  • Фильтрующие показатели:
    • Организация – основная организация компании. Если в системе Организация одна, то по умолчанию она в документе скрыта.
    • Вид перевозки – устанавливается вид перевозки для подбора задания на перевозку груза в рейс;
    • Заказчик – устанавливается партнер-заказчик перевозки;
    • Отправитель – устанавливается партнер-отправитель груза;
    • Получатель – устанавливается партнер-грузополучатель груза;
    • Зоны и Адреса отправки/получения грузов – устанавливаются зоны и адреса отправления/получения груза для периодического рейса;
  • Редактировать плановое расписание – устанавливается расписание с указанием Даты начала рейса, Количества ТС, Времени начала рейса. На форме планирования периодического рейса графически отображаются установленные расписания.
