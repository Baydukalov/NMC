Справочник "Соглашения с заказчиком"

Условия сотрудничества, а также договорные условия работы с заказчиками регистрируются в справочнике Соглашения с заказчиком. Справочник доступен при подключении модуля Управление перевозками. Справочник Соглашения с заказчиком может быть открыт при помощи одноименной команды панели навигации раздела Нормативно-справочная информация из группы Партнеры.

В соглашении указывается:

  • Наименование и Период действия соглашения.

  • Организация – собственное юридическое лицо, от имени которого будет осуществляться перевозка по этому соглашению. Если в системе используется только одна Организация, то она по умолчанию скрыта с формы.

  • Группа тарифов, которая будет использоваться при тарификации услуг по этому соглашению, а также определяет валюту расчетов.

  • Макс % скидки – который может использоваться при тарификации услуг по этому соглашению. Контроль осуществляется при оформлении документов Заявка на перевозку и Фактические доходы.

  • Заказчик – партнер, которому будут оказываться услуги перевозки.

  • Контрагент партнера, на которого оформляется пакет документов по этому соглашению.

  • Контактное лицо партнера, с которым можно контактировать при выполнении условий этого соглашения.

В табличной части на закладке Статьи доходов указывается перечень статей доходов по умолчанию, которые могут использоваться при оформлении Заявок на перевозку груза по этому соглашению.
