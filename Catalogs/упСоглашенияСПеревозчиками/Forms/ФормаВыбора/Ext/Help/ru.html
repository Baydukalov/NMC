Справочник "Соглашения с перевозчиком"

Условия сотрудничества, а также договорные условия работы с перевозчиками регистрируются в справочнике Соглашения с перевозчиком. Справочник доступен при подключении модуля Управление перевозками. Справочник Соглашения с перевозчиком может быть открыт при помощи одноименной команды панели навигации раздела Нормативно-справочная информация из группы Партнеры.

В соглашении указывается:

  • Наименование и Период действия соглашения.

  • Статус соглашения - Действует или Закрыто.

  • Организация – собственное юридическое лицо, для которого будет осуществляться перевозка по этому соглашению. Если в системе используется только одна Организация, то она по умолчанию скрыта с формы.

  • Группа тарифов, которая будет использоваться при тарификации услуг по этому соглашению, а также определяет валюту расчетов.

  • Перевозка осуществляется транспортной компанией регулирует возможность осуществления перевозки транспортной компанией целиком.

  • Перевозчик – партнер, который будет осуществлять перевозку.

  • Контрагент партнера, от которого будет получен пакет документов по этому соглашению.

  • Контактное лицо партнера, с которым можно контактировать при выполнении условий этого соглашения.

В табличной части на закладке Статьи расходов указывается перечень статей расходов по умолчанию, которые могут использоваться при получении предложений от перевозчиков и при оформлении рейсов по этому соглашению.
