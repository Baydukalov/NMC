Справочник "Работы по ремонту"

Перечень работ, выполняемых в рамках ремонтных работ, хранится в справочнике Работы по ремонту.

Справочник Работы по ремонту доступен только при подключении модуля Управление транспортом и может быть открыт при помощи одноименной команды панели навигации раздела Сервис и настройки из группы ГСМ и ремонты.

Для каждой работы указывается наименование и модель транспортного средства, для которой выполняется эта работа,плановое время работ. При помощи команды Цены работ по ремонту можно перейти к интерфейсу для ввода информации о ценах. Информация о ценах является периодической и хранится в регистре сведений Цены работ по ремонту.

Элементы справочника Работы по ремонту используются при заполнении табличной части на закладке Работы документа Ремонт (ТО).
