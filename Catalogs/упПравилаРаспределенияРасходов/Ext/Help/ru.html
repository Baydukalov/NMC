Справочник "Правила распределения расходов"

Для расчета финансовых показателей эффективности деятельности предприятия в системе по каждой заявке может быть рассчитан валовый доход. Для этой цели доходы и расходы должны быть распределены по документам Заявка на перевозку. При детализации груза по грузовым местам, расходы могут быть дополнительно распределены по грузовым местам, что позволит получить более точную оценку стоимости услуг перевозки для формирования плановых цен на услуги. Для хранения правил распределения расходов в системе используется справочник Правила распределения расходов. 

Для каждого правила указывается наименование и способ распределения. Перечень дополнительных параметров, которые необходимо указать для правила определяются выбранным способом распределения. Если для правила выбран способ распределения Пропорционально параметру, то в качестве дополнительных параметров указывается только показатель, формирующих базу для распределения. В системе могут быть использованы следующие показатели:

  • Время (сек);

  • Расстояние;

  • Масса;

  • Объем;

  • Количество грузовых мест;

  • Количество загрузочных метров;

  • Плановые расходы по рейсу;

  • Оценочная стоимость груза.

При выборе способа распределения Указать формулу распределения для правила описывается формула для распределения с использованием синтаксиса языка 1С:Предприятие. В формуле могут использоваться функции, приведенные в разделе Функции, и показатели, из раздела Показатели. Перечень функций и показателей ограничен средствами языка 1С:Предприятие и возможностями системы.

Полученная формула будет отображаться в реквизите Формула расчета и может быть скорректирована пользователем вручную.
