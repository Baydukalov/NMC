Справочник "Стили линий"

Стили линий используются при формировании цветовой схемы и используются для отображения на электронной географической карте линий маршрута рейса. Перечень стилей линий храниться в одноименном справочнике.

Справочник Стили линий доступен при подключении модулей Управление перевозками и может быть открыт при помощи одноименной команды панели навигации раздела Сервис и настройки из группы Электронные географические карты.

Для каждого стиля указывается:

  • Наименование, понятное пользователям системы.

  • Цвет границы, который будет использоваться при формировании линии.

  • Толщина границы, которая будет использоваться при формировании линии.

  • Прозрачность границы, которая будет использоваться при формировании линии.
