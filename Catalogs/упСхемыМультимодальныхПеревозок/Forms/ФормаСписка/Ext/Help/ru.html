Справочник "Схемы мультимодальных перевозок"

Справочник Схемы мультимодальных перевозок используется для формирования маршрута выполнения заявок. При выборе в заявке схемы мультимодальной перевозки в реквизите Цепь поставок, маршрут перевозки и количество заданий, создаваемых по этой заявке, будут определятся параметрами выбранной схемы.

Справочник Схемы мультимодальных перевозок доступен только при подключении модуля Управление перевозками и может быть открыт при помощи одноименной команды панели навигации раздела Сервис и настройки из группы Настройки планирования.

Схема мультимодальной перевозки может состоять из нескольких этапов. Перечень этапов и их добавление выполняется на закладке Схема маршрута.

Для каждого этапа указываются параметры перевозки и транзитный адрес. Параметры перевозки по умолчанию используются при формировании заданий на перевозки для выполнения заявки с мультимодальной схемой перевозки. На закладке Дополнительно указываются параметры отбора при помощи которых задаются ограничения использования конкретной схемы мультимодальной перевозки в заявке. В качестве параметров указываются вид перевозки, организация, принявшая заявку, заказчик и соглашение с заказчиком.

Реквизит Время в пути/Время в пути до конечной точки, учитывается при расчете этапов перевозки при переводе статуса Заявки на перевозку груза из Новая в К выполнению. Если по установленным данным при расчете времени в пути до конечной точки, временное окно получения груза, меньше рассчитанного системой, то статус заявки не изменится.

На закладке Настройки генерации заданий указываются настройки по заполнению временных окон в заданиях мультимодальной схемы.

  • Определять временные окна заданий – при включенном реквизите происходит заполнение временных окон в Заданиях на перевозку груза согласно доступным вариантам, установленным в Параметрах заполнения перевозки (по умолчанию).
    • Формировать временные окна по графикам работы адресов. При включенной опции при формировании заданий на перевозку заполняются временные окна согласно установленным Параметрам:
      • Варианты заполнения временных окон в заданиях указываются в Параметрах заполнения перевозки(по умолчанию):
        • Постоянная доступность – временное окно не заполняется;
        • Полное ограничение – во временном окне заполняются обе даты «с» и «по»;
        • Только начало окна – заполняется только дата «с»;
        • Только окончание окна – заполняется только дата «по».
        • В парметрах заполнения перевозки также указывается реквизит Время нахождения в точке. Задает дополнительное время, которое тратится на точку. Установленное значение увеличивает даты «Отправления с … по» из этой точки.
