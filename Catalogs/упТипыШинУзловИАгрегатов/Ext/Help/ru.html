Справочник "Типы шин, узлов и агрегатов"

Для каждого типа шины, узла или агрегата указывается:

  • Наименование, как правило по каталогу;

  • Модель транспортного средства, для которой может использоваться этот тип;

  • На закладке Основное указывается:

    • Вид типа шины, узла, агрегата, доступный из выбора: Шины, Аккумулятор, Прочее;
      • Если Вид типа шины, узла, агрегатов указан как Шины, то возможно указать: Высоту, Ширину, Профиль, Сезонность резины.

    • Фиксировать срок полезного использования, при включенном реквизите списание затрат происходит на косвенные расходы или расходы будущих периодов по сроку полезного использования, иначе списание затрат осуществляется по норме при помощи путевого листа.
      • Срок полезного использования (в месяцах), устанавливается срок полезного использования типа агрегата.

    • Норма эксплуатационного ресурса, указывается либо в километрах, либо в моточасах;

    • Признак Измерять норму в моточасах, возможно использовать, например, при работе оборудования;

    • Статья расходов, указывается статья для отнесения выработки агрегатов и Комментарий.

  • На закладке Дополнительно указывается:

    • Состояние оборудования (по умолчанию), которое может принимать значение Не используется, Оприходовано на склад, Установлено на ТС, Списано.

    • Комментарий.
