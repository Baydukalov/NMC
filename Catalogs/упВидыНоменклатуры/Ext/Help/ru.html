Справочник "Виды номенклатуры"

Справочник Виды номенклатуры используется для обобщенной классификации номенклатуры по общим учетным принципам.

Например,

к конкретному виду номенклатуры могут относится номенклатурные позиции:

  • с одинаковым назначением: кондиционеры, пылесосы и т.д.;

  • для которых используется учет по характеристикам с едиными принципами ее формирования: для обуви могут использоваться характеристика цвет и размер, а для одежды ткань и размер;

  • для которых используется единая структура упаковок: для обуви может использоваться пара – короб – паллета, а для сыпучих продуктов может использоваться килограмм – тонна.

Перечень видов номенклатуры хранится в справочнике Виды номенклатуры.

Для каждого вида номенклатуры указывается Наименование, Набор свойств, как дополнительных параметров номенклатуры этого вида, и справочно уточняются типы кузовов для погрузочно-разгрузочных работ.
