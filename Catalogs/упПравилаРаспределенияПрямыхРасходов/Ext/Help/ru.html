Справочник "Правила распределения прямых расходов"

Справочник Правила распределения прямых расходов служит для формирования правил распределения расходов по выполненным рейсам, транспортным средствам и направлениям деятельности.

Для каждого Правила распределения прямых расходов указывается какие расходы участвуют в распределении:

■        Расходы по рейсам;

■        Расходы на транспорт;

■        Финансовый результат – расходы по направлениям деятельности.

На закладке Распределение расходов указывается Вариант распределения, который по заданным параметрам отбора отбирает необходимые расходы для дальнейшего распределения.

На закладке Варианты распределения указываются Варианты распределения по приоритетности использования. Согласно установленному Варианту распределения распределяются расходы согласно установленным параметрам. Нераспределенные остатки расходов распределяются по следующему Варианту распределения согласно списку до момента окончания нераспределенных остатков расходов или списка Вариантов расходов.
