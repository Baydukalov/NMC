Справочник "Контактные лица партнеров"

Справочник Контактные лица партнеров доступен при помощи одноименной команды из формы элемента справочника Партнеры.

Для каждого контактного лица указывается фамилия, имя, отчество, должность, дата рождения, пол и контактная информация на закладке Адреса, телефоны. Данные контактного лица используются для организации коммуникаций с партнером.

Кроме этого, в информационной базе может храниться связанная с партнером информация о подразделениях партнеров, контактных лицах партнеров, взаимодействиях и соглашениях, которая используется при вводе документов системы.
