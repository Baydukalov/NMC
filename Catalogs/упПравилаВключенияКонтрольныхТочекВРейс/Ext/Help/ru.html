Правила включения контрольных точек в рейс

Правила включения контрольных точек в рейс используется в процессе планирования рейсов и корректировки маршрута с учетом проезда через адреса, требующие дополнительного проезда через контрольные точки.

Для каждого правила заполняется следующая информация:
  • Наименование Правила включения контрольных точек в рейс понятное Пользователю;
  • Признак Активно включается для использования данного правила при автоматической корректировки маршрута;
  • Для ограничения периода действия плавила заполняется Период действия;

Для отбора Рейсов для автоматического включения контрольных точек возможно использовать отборы в группе Быстрый отбор.

На закладке Контрольные точки заполняется список контрольных точек для включения в Рейсы.

На закладке Отбора по адресам заполняются настройки отбора по начальным и конечным адресам Рейса:

При включенном параметре Отбирать по начальной точке Пользователю доступен выбор показателя:

  • Адрес – заполняется Адрес, после которого будет добавлена контрольная точка в маршрут Рейса
  • Вид адреса – заполняется Вид адреса, после которого будет добавлена контрольная точка
  • Зона – заполняется Географическая зона, после которой будет добавлена контрольная точка

При включенном параметре Отбирать по конечной точке Пользователю доступен выбор показателя:
  • Адрес – заполняется Адрес, перед которым будет добавлена контрольная точка в маршрут Рейса
  • Вид адреса – заполняется Вид адреса, перед которым будет добавлена контрольная точка
  • Зона – заполняется Географическая зона, перед которой будет добавлена контрольная точка

На закладке Дополнительно устанавливаются дополнительные настройки включения контрольных точек:
  • Параметр Сдвигать время начала рейса на разницу от изменения маршрута – позволяет корректировать время начала рейса с учетом сдвига маршрута по установленной контрольной точки.
  • Параметр Запрещено многократное применение правила к рейсу устанавливает запрет на повторное добавление контрольных точек в рейсы.
