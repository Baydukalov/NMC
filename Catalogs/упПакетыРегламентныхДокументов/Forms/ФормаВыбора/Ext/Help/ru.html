Справочник "Пакеты регламентных документов"

Контроль возврата документов может осуществляется по каждому партнеру. При этом для отражения в учете услуг факта передачи и получения грузов партнер может использовать не один, а несколько типов документов. Для этого в системе для каждого партнера может быть задан свой индивидуальный комплект документов и адрес для возврата документов партнеру.

Пакеты документов, которые могут использоваться партнерами хранятся в справочнике Пакеты регламентных документов. Справочник Пакеты регламентных документов может быть открыт при помощи одноименной команды панели навигации раздела Сервис и настройки из группы Учет документов.

Для каждого пакета документов указывается наименование и перечень типов документов, входящих в пакет. При описании перечня документов, входящих в пакет, для каждого типа документа указывается:

  • Участник – роль участника процесса перевозки, который передает или получает документ и может принимать значения Грузоотправитель, Грузополучатель или Заказчик.

  • Получить (кол. экземпляров) – количество экземпляров этого типа документов которое необходимо получить у участника процесса. При этом для точки маршрута в рейсе будет создана дополнительная операция, связанная с получением документов.

  • Передать (кол. экземпляров) - количество экземпляров этого типа документов которое необходимо передать участнику процесса. При этом для точки маршрута в рейсе будет создана дополнительная операция, связанная с передачей документов.

  • Возврат документов после исполнения рейса – флаг необходимости возврата документов по адресу партнера, выбранного в задании и выполняющего указанную роль участника процесса перевозки. При установке этого флага в маршрут рейса добавляется точка маршрута, связанная с возвратом документов по адресу партнера.
