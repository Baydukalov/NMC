Справочник "Виды адресов"

Для каждого географического адреса должен быть указан Вид адреса, который влияет на параметры отображения адреса на электронной географической карте и для контроля корректности указания адресов в задании на перевозку при планировании рейсов. Перечень видов адресов храниться в одноименном справочнике.

Для каждого вида адреса указывается:

  • Наименование понятное пользователям системы.
  • Время прохождения точки, как плановое прохождение адресов-точек маршрута этого вида.
  • При использовании функций автоматического расчета расстояний необходимость рассчитывать расстояния по такому виду адресов в первую очередь при помощи флага Приоритетное построение массива расстояний.
  • Признак Не использовать как базисный адрес отправления выставляется для Вида адреса в случае, если в при формировании Рейса участвуют Задания на перевозку груза только на возврат, а адрес получаения соответствует адресу отправления Вида перевозки.
  • Стили маркера, выделенного маркера и выбранного маркера для отображения адресов этого вида на электронной географической карте.
  • Стиль вершины графа- стиль вершины сетевого графа
