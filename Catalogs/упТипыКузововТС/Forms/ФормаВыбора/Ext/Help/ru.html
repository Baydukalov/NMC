Справочник «Типы кузовов ТС для погрузки»

Справочник Типы кузовов ТС для погрузки предназначен для хранения типов кузовов ТС, используемых компанией для перевозки грузов.

Например: боковой борт, задние ворота и т.д

Справочник вызывается из меню «Справочники» - «Типы кузовов для погрузки».

Справочник имеет одноуровневую структуру. Редактирование и ввод новых элементов производится непосредственно в форме списка справочника. При вводе нового элемента справочника указываются только код и наименование.

Элементы справочника используются при описании моделей транспортных средств.
