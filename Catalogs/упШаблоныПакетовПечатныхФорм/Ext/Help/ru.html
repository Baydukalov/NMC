Справочник "Шаблоны пакетов печатных форм"


В справочнике Шаблоны пакетов печатных форм задается количество печатающихся документов для конкретного партнера, которые используются при пакетной печати из рейса. Шаблон используется для точечной настройки количества печатаемых экземпляров документов с точностью до получателя груза. Пакеты печатных форм по нескольким заданиям в рейсе выводятся в соответствии с порядком отгрузки. Пакетная печать вызывается из рейса по командам:

  • Пакет печатных форм по рейсу;

Пакет печатных форм по рейсу (на принтер) – печать пакета документов осуществляется сразу на принтер, без предварительного просмотра. Учет пакетов печатных форм с точностью до экземпляров документов может быть осуществлен:

  • По Рейсу;

  • По Заданию на перевозку груза.

В шаблоне пакетов печатных форм с установленным Видом документа в значении Рейс по команде Заполнить доступны печатные формы, которые могут быть распечатаны на основании рейса.

Например,
Шаблон пакета печатных форм по Рейсу устанавливается у партнера-перевозчика, которому для исполнения рейса необходимо распечатать Маршрутный лист.

В шаблоне пакетов печатных форм, где Вид документа установлен в значение Задание на перевозку груза, по команде Заполнить доступны печатные формы, которые возможно распечатать на основании Задания на перевозку груза.

В шаблоне заполняется:

  • Наименование - понятное пользователю;

  • Вид документа - Рейс или Задание на перевозку груза;

  • Разбирать по копиям-при выводе на печать, документы печатаются пачками, согласно установленному количеству экземпляров;

  • Табличная часть:

    • Представление макета – устанавливаются необходимые печатные формы;
    • Порядок печати печатной формы при выводе на принтер;
    • Количество – количество экземпляров печатной формы необходимое для в пакете партнера;
    • Количество вернуть – количество экземпляров печатных форм документов, которые необходимо вернуть по факту исполнения рейса;
    • Тип регламентного документа – соответствие макета и типа регламентного документа с учетом способов определения номера и даты экземпляра. При возврате документов по рейсу согласно установленному способу определения номера и даты экземпляра документов производиться сверка возвращенных экземпляров.
