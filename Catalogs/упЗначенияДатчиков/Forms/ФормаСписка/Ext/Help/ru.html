Справочник "Значения датчиков"

Значения датчиков, которыми оснащены трекеры, установленные на транспортных средствах предприятия, хранятся в справочнике Значения датчиков.

Справочник Значения датчиков может быть открыт при помощи одноименной команды панели навигации раздела Сервис и настройка из группы Мобильные устройства и трекеры.

Для каждого датчика указывается значение и, при необходимости вид ГСМ, уровень которого контролируется этим датчиком.
