Справочник "Правило доступности географических зон"

Ограничение использования транспортных средств в определенных географических зонах регулируется при помощи правил доступности географических зон. Правило доступности географических зон указывается для каждого транспортного средства индивидуально. Перечень правил хранится в справочнике Правило доступности географических зон.

Справочник Правило доступности географических зон доступен только при подключении модуля Управление перевозками и может быть открыт при помощи одноименной команды панели навигации раздела Сервис и настройки из группы Топология.

Для каждого правила указывается перечень географических зон характер правила (запрет или доступность) при помощи флага Запрещена работа в указанных зонах. Если флаг установлен, то правилом определяется перечень запрещенных зон, если снят – то доступных.

В справочнике содержится предопределенный элемент Доступны все зоны, который используется по умолчанию при вводе нового транспортного средства.
