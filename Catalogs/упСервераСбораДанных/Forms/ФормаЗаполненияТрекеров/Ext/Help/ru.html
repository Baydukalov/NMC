Мастер сопоставления трекеров сервера сбора данных и объектов конфигурации TMS Управление транспортом и перевозками.

Для сопостовления выполните следующие шаги:

1) Выполните команду "Заполнить с сервера", после загрузки данных нажмите далее.

2) На странице сопостовления трекеров отметьте строки для которых нужно найти уже созданные трекеры в базе, выполните команду "Найти трекеры".

3) Для создания трекеров, отметьте строки, укажите шаблон имени трекера, модель трекера выполните команду "Создать трекеры".

4) На странице установке трекеров на транспортные средства отметьте строки для трекеров которых нужно установить транспортное средство, в поле "Дата установки" укажите дату с которой будет установлен трекер на транспортное средство, выполните команду "Установить трекеры на транспортные средства"

5) для завершения работы мастера сопостовления трекеров, выполните команду "Завершить"
