Справочник "Сервера сбора данных"

Перечень серверов, которые используются для сбора данных с трекеров, установленных на транспортных средствах хранится в справочнике Сервера сбора данных.

Для каждого сервера указывается его тип и параметры. На текущий момент в системе могут использоваться два типа сервера 1С: Центр спутникового мониторинга, AutoGRAPH и Android клиент. Для выбранного сервера указываются параметры подключения к нему в реквизитах Сервер и Порт. По факту ввода этих данных в системе генерируется двумерный штрихкод, который можно использовать для подключения к серверу сбора данных. При установке признака Создавать события мониторинга на основе данных, полученных с сервера сбора данных, выполняется создание событий мониторинга. При выборе типа сервера AutoGRAPH указываются Схема передачи данных при помощи AutoGRAPH.NET Service и Дата начала загрузки данных – дата начала передачи данных с сервера сбора данных.
