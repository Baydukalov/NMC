Справочник "Модели транспортных средств"

Модели транспортных средств в системе используются для классификации транспортных средств по общим эксплуатационным характеристикам на конкретном предприятии. Справочник Модели транспортных средств доступен при приобретении модулей Управление перевозками и Управление транспортом. Справочник Модели транспортных средств может быть открыт при помощи одноименной команды панели навигации раздела Нормативно-справочная информация из группы Транспортные средства. Модель транспортного средства в системе может вводиться на основании элемента справочника Типы транспортных средств. В этом случае весогабаритные характеристики заполняются на основании данных выбранного типа транспортных средств.

Кроме весогабаритных характеристик для каждой модели транспортных средств указываются:  

  • На закладке Прочие характеристики указываются:
    • Печатная форма путевого листа – используется для печати формы путевого листа для выбранной модели ТС. Выбор формы путевого листа доступен только при подключении модуля Управление транспортом.
    • Допустимая нагрузка на ось, как справочная информация.
    • Объема бака модели ТС – используется для контроля перелива топлива при осуществлении заправок. Форма доступна только при подключении модуля Управление транспортом.
  • На закладке Статьи расходов необходимо указать перечень статей расходов, с использованием которых будут по умолчанию формироваться расходы по рейсу, выполняемому транспортным средством, и относящимся к этой модели.
  • Дополнительное устанавливаемое оборудование для модели транспортных средств указывается на закладке Устанавливаемое оборудование. Список оборудования формируется выбором элементов из справочника Дополнительное оборудование. В качестве дополнительного оборудования может быть установлен манипулятор грузового транспортного средства и другое аналогичное оборудование. Для каждого оборудования, выбранного в табличной части, указывается время монтажа и демонтажа оборудования в часах, которое используется при планировании рейсов.
  • Нормативы расхода ГСМ и технических жидкостей, которые в дальнейшем будут использоваться при исполнении рейсов для транспортных средств и относящих к этой модели, указываются на закладке Нормы расхода. При этом в табличной часть указываются основной ГСМ, прочие ГСМ и технические жидкости. Дата установки – указывается дата, с которой начинается применение установленной Нормы расхода.
  • На закладке Комментарий указывается дополнительная информация о модели ТС.

На основании модели транспортного средства можно создать новое транспортное средство.
