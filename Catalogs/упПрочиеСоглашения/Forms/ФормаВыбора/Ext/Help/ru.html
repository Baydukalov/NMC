Справочник "Прочие соглашения"

Для регистрации условий сотрудничества с партнерами, не связанными с предоставлением и получением услуг по перевозке, в системе используется справочник Прочие соглашения. Справочник Прочие соглашения может быть открыт при помощи одноименной команды панели навигации раздела Нормативно-справочная информация из группы Партнеры.

В соглашении указывается:

  • Наименование и Период действия соглашения.

  • Статус соглашения - Действует или Закрыто.

  • Тип соглашения используется для отбора соглашений в документах системы и может принимать одно значений: Соглашение с сервисной организацией, Соглашение с АЗС, Соглашение со страховщиком, Прочее.

  • Организация – собственное юридическое лицо, для которого будет осуществляться оказание прочих услуг по этому соглашению. Если в системе используется только одна Организация, то она по умолчанию скрыта с формы.

  • Группа тарифов, которая будет использоваться при тарификации услуг по этому соглашению, а также определяет валюту расчетов.

  • Партер, который будет осуществлять оказание услуг.

  • Контрагент партнера, от которого будет получен пакет документов по этому соглашению.

  • Контактное лицо партнера, с которым можно контактировать при выполнении условий этого соглашения.

В табличной части на закладке Статьи расходов указывается перечень статей расходов по умолчанию, которые могут использоваться при оформлении документов Ремонт, Прочие расходы, Страхование грузов по этому соглашению.
