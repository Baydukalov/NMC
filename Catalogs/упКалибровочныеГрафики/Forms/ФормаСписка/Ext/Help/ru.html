Справочник "Калибровочные графики"

Для интерпретации данных полученных с датчиков в значения, которые могут использоваться в системе, используются калибровочные графики.

Справочник Калибровочные графики может быть открыт при помощи одноименной команды панели навигации раздела Сервис и настройка из группы Мобильные устройства и трекеры.

В калибровочном графике описывается правило для интерпретации значения полученного с датчика, в пределах указанного входного диапазона, в конкретное значение датчика, которое в дальнейшем может использоваться при регистрации событий мониторинга. События мониторинга по интерпретированным значениям датчиков, полученным при помощи калибровочных графиков, могут отображаться на карте в рабочем месте РМ: Спутниковый мониторинг.

Например

Необходимо отобразить факт остановки двигателя в рабочем месте РМ: Спутниковый мониторинг. Для этого создается новый тип события спутникового мониторинга и указывается какое интерпретированное значение датчика должно быть зарегистрировано в системе для того, чтобы это событие отобразилось в рабочем месте РМ: Спутниковый мониторинг. По факту получения с трекера значения датчика контроля состояния двигателя «0» в системе будет зарегистрировано событие Остановка двигателя, которое будет отображаться на карте в рабочем месте РМ: Спутниковый мониторинг при помощи черной точки по месту дислокации транспортного средства.
