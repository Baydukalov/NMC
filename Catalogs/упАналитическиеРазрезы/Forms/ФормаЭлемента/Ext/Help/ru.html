Справочник "Аналитические разрезы"

Аналитические разрезы используются для детализации расходов с детализацией до грузовых мест. При задании отдельных аналитических разрезов возможно распределить расходы по группам товаров с дальнейшим анализом затрат в отчете Анализ расходов по рейсам. Состав параметров аналитического разреза используется также при интеграции программами для ведения бухгалтерского учета.

Справочник Аналитические разрезы доступен только при подключении модуля Управление перевозками и может быть открыт при помощи одноименной команды панели навигации раздела Сервис и настройки из группы Тарификация.

Для каждого элемента справочника указывается наименование партнер, подразделение партнера и комментарий.
