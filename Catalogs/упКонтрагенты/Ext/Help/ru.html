Справочник "Контрагенты"

В списке связанных контрагентов при помощи кнопки Установить основным выделяется основной контрагент, который в дальнейшем будет использоваться по умолчанию в документах системы.

Для каждого контрагента указывается:

  • Сокращенное юр.наименование, которое будет использоваться в печатных формах документов.

  • Рабочее наименование, которое будет понятно пользователям системы.

  • Признак Юр/Физлицо при помощи которого регулируется доступность реквизитов, указываемых для контрагента. Например, для юридического лица будут доступны ИНН, КПП, код по ОКПО, а для Физлица только ИНН.

  • ИНН, КПП, Код по ОПКО, доступность которых зависит от вида контрагента.

Основной контрагент выделяется флагом Основной.

На закладке Адреса, телефоны указывается вся необходимая контактная информация, которая будет использоваться при формировании печатных форм.
