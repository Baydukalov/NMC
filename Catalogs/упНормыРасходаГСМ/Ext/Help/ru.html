Справочник "Нормы расхода ГСМ"

Непосредственно нормы расхода ГСМ, которые указываются для моделей транспортных средств, хранятся в справочнике Нормы расхода ГСМ. 

Для каждой нормы указывается:

  • Тип нормы при помощи переключателя, который может принимать значение на 100 км или на 1 час работы.

  • Расход – непосредственно норма расходы ГСМ.

  • Изменение расхода на перевозку полезного груза.

  • Надбавки - в табличной части возможно указать либо процент от базовой нормы, либо количество литров на один час работы, количество литров на 100 километров, либо количество литров на одну операцию.

  • Шкала температурных коэффициентов - для расчета норм расхода ГСМ, которая выбирается из справочника Шкалы температурных коэффициентов.

  • Формула расчета – для расчета норм расхода ГСМ, которая выбирается из справочника Правила расчета норм. Если формула не указана, то при расчет нормативного потребления ГСМ используется формула по умолчанию заложенная в системе заложенная в системе: Qн = 0,01 * (Hs·S + Hw·W), где Hs·- норма расхода, S – пройденное расстояние, Hw – изменение расхода, W – масса груза.

Элементы справочника Нормы расхода ГСМ используются при определении нормативного расхода ГСМ в справочнике Модели ТС.
