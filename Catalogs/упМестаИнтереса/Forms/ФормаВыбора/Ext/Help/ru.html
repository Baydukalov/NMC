Справочник "Места интереса"

При помощи Мест интереса в РМ: Планирование рейсов возможно вывести на карту места, стандартно не отображаемые на картах по умолчанию.

Справочник Места интереса доступен только при подключении модуля Управление перевозками и может быть открыт при помощи одноименной команды панели навигации раздела Сервис и настройки из группы Топология.

Для каждого места интереса указывается:

  • Наименование понятное пользователям системы.

  • Родитель - определяет подчиненность места в иерархии справочника.

  • Стиль маркера - позволяет определить цвет выделения места интереса на электронной географической карте.

  • Табличная часть с указанием адресов мест интереса.
