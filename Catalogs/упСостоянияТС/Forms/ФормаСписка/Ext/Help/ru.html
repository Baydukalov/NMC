Справочник «Состояния транспортных средств»

Справочник Состояния транспортных средств доступен только при приобретении модуля Управление перевозками.

Перечень возможных состояний транспортных средств предприятия указывается в справочнике Состояния транспортных средств. Справочник содержит предопределенные элементы, которые устанавливаются автоматически при возникновении определенных событий в системе. Использование состояний транспортного средства указывается в настройках системы.

Справочник Состояния транспортных средств может быть открыт при помощи одноименной команды панели навигации раздела Нормативно-справочная информация из группы Транспортные средства.

При необходимости перечень состояний может быть расширен пользователями самостоятельно.
