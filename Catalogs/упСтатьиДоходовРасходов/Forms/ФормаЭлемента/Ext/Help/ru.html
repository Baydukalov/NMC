Справочник "Статьи доходов и расходов"

В справочнике Статьи доходов и расходов хранится перечень статей, которые используются для регистрации и ведения аналитического учета доходов и расходов.

При помощи переключателя Тип статьи указывается назначение статьи, т.е. для чего она используется для планирования и регистрации доходов или расходов. Тип статьи используется для отбора при выборе этой статьи в документах системы и влияет на видимость и доступность реквизитов статьи.

Для каждой статьи указывается наименование, ее место в иерархии справочника и полное наименование, которое будет использоваться при формировании печатных форм.

На закладке Параметры для статьи расходов необходимо указать:

  • Вариант расходов:
    • Прямые. Расходы, которые возможно отнести к рейсу.
    • Косвенные. Расходы, которые нельзя однозначно отнести к рейсу.
    • Расходы будущих периодов. Расходы, которые списываются с распределением на будущие отрезки времени в соответствии с установленными правилами.
  • Правило распределения расходов по заданиям на перевозку.
  • Ставка НДС.

Также возможно указать Код счета регламентированного плана счетов, Вид расхода – вид расхода налогового учета: Амортизацию, Командировочные расходы, Оплату труда, Прочие расходы, Ремонт основных средств, Транспортные расходы. Код счета и Вид расходов используются для интеграции с учетными бухгалтерскими системами.

Описание порядка использования статьи в системе можно указать на закладке Описание.
