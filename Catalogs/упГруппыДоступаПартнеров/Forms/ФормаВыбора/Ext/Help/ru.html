Справочник "Группы доступа партнеров"

Для группировки партнеров по каким-либо правилам, удобным для конкретного предприятия, используется справочник Группы доступа партнеров. Справочник Группы доступа партнеров может быть открыт при помощи одноименной команды панели навигации раздела Нормативно-справочная информация из группы Партнеры. Справочник не является иерархическим и позволяет организовать только один уровень иерархии для элементов справочника Партнеры.
