Справочник "Запчасти"

При выполнении ремонтных работы могут использоваться запасные части. Для хранения перечня запасных частей, используемых при выполнение ремонтных работ, в системе используется справочник Запчасти.

Справочник Запчасти доступен только при подключении модуля Управление транспортом и может быть открыт при помощи одноименной команды панели навигации раздела Сервис и настройки из группы ГСМ и ремонты.

Для каждой запчасти указывается наименование, как правило по каталогу, единица измерения, модель транспортного средства, для которой может использоваться эта запчасть.

При помощи команды Цены запчастей можно перейти к интерфейсу для ввода информации о ценах. Информация о ценах является периодической и хранится в регистре сведений Цены запчастей.

Элементы справочника Запчасти используются при заполнении табличной части на закладке Запчасти документа Ремонт (ТО).
