Правила подбора перевозчиков на рейсы

Для каждого правила заполняется следующая информация:
  • Родитель – элемент, стоящий выше по иерархии
  • Наименование Правила подбора перевозчиков на рейсы понятное Пользователю;
  • Дата окончания действия – срок, в рамках которого производится распределение рейсов на перевозчиков;
  • Период анализа данных – период, в рамках которого производится общее распределение перевозчиков на рейсы. Доступны следующие периоды анализа:
    • День начала рейса – распределение Рейсов по Партнерам- перевозчикам в рамках одного дня;
    • Неделя начала рейса - распределение Рейсов по Партнерам- перевозчикам в рамках одной недели;
    • Месяц начала рейса - распределение Рейсов по Партнерам- перевозчикам в рамках одного месяца.

Для отбора Рейсов для подбора и распределения перевозчиков возможно использовать отборы в группе Быстрый отбор и на закладке Настройки отбора рейса.

На закладке Перевозчики заполняется распределение по Перевозчикам, Контрагентам и Соглашению в процентном соотношении.

На закладке Дополнительно заполняется Важность для вида правила и дополнительный Комментарий.
