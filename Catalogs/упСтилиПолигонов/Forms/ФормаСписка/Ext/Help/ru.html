Справочник "Стили полигонов"

Стили полигонов используются при формировании цветовой схемы и используются для отображения на электронной географической карте географических или прочих зон. Перечень стилей полигонов хранится в одноименном справочнике.

Справочник Стили полигонов доступен при подключении модулей Управление перевозками и может быть открыт при помощи одноименной команды панели навигации раздела Сервис и настройки из группы Электронные географические карты.

Для каждого стиля указывается:

  • Наименование, понятное пользователям системы.

  • Цвет границы, который будет использоваться при формировании границы зоны.

  • Толщина границы, который будет использоваться при формировании границы зоны.

  • Прозрачность границы, которая будет использоваться при формировании границы зоны.

  • Цвет заливки, который будет использоваться при формировании заливки зоны.

  • Прозрачность заливки, которая будет использоваться при формировании заливки зоны.
