Справочник "Типы регламентных документов"

Для контроля получения, передачи и возврата документов при получении или передачи грузов, в системе ведется учет документов по типам.

Для каждого типа документа указывается наименование и префикс штрихкода, используемый при сканировании и распознавании документов, указывается способ определения номера экземпляра, способ определения даты экземпляра, в случае оформления экземпляров документов.

Для автоматического формирования Экземпляров документов в справочнике Типы регламентных документов используются следующие реквизиты:
  • Способ определения номера экземпляра:
    • Номер – указывается номер документа основания
    • Номер КИС – указывается номер КИС документа основания (задания на перевозку) или если он не заполнен, то указывается номер КИС заявки на перевозку груза
    • По формуле – для формирования номера экземпляра используется формула
  • Способ определения даты экземпляра:
    • Дата – дата документа основания
    • Дата КИС – дата КИС документа основания
    • По формуле – для формирования даты используется формула.
