Справочник "Варианты распределения расходов"

 

Справочник Варианты распределения расходов предназначен для задания правил распределения расходов по установленным настройкам и отборам.

Для Варианта распределения расходов указывается:

■        Наименование;

■        Правило распределения (подробнее см. Варианты распределения расходов);

■        Вариант расходов (прямые, косвенные);

■        Алгоритм отборов. Доступны следующие алгоритмы:

o    Настройка отбора – устанавливаются пользовательские настройки;

o    Произвольный алгоритм - в этом случае в появившемся текстовом поле на языке 1С:Предприятие 8.3. может быть написан произвольный алгоритм.

Расходы могут быть распределены по Рейсам по следующим показателям расходов:

■        Аналитический разрез – распределяются расходы с заполненным Аналитическим разрезом по рейсам с таким же Аналитическим разрезом;

■        Направление деятельности – распределяются расходы с заполненным Направлением деятельности по рейсам с таким же Направлением деятельности;

■        Подразделение – распределяются расходы с заполненным Подразделением по рейсам с таким же Подразделением;

■        Организация – распределяются расходы по Организациям.

Дополнительно в поле Настройка отбора рейсов для распределения доступны дополнительные отборы Рейсов.
