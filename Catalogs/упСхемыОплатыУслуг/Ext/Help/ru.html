Справочник "Схемы оплаты услуг"

Схемы оплаты услуг, которые используются для контроля перехода статусов документов Заявка на перевозку груза и Рейс на основании оплаченных счетов, хранятся в справочнике Схемы оплаты услуг.

Для каждой Схемы оплаты услуг указываются:

·         Наименование;

·         В ТЧ Вариант оплаты по этапу необходимо указать процент оплаты по каждому из представленных этапов работы процесса перевозки груза.

Примечание

В случае, если нет необходимости производить оплату по одному из этапов, то оставить соответствующее поле пустым.
