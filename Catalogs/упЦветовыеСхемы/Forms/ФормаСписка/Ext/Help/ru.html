Справочник "Цветовые схемы"

Цветовые схемы используются для отображения на электронной географической карте маршрутов рейса, адресов и позволяют определить параметры отображения линий и точек. Перечень цветовых схем храниться в одноименном справочнике.

Справочник Цветовые схемы доступен только при подключении модуля Управление перевозками и может быть открыт при помощи одноименной команды панели навигации раздела Сервис и настройки из группы Электронные географические карты.

Для каждой схемы указывается:

  • Наименование понятное пользователям системы.

  • Возможность использования схемы при отображении маршрута рейса при помощи флага Использовать для отображения маршрута.

  • Приоритет использования схемы, который будет использоваться при отображении маршрута рейса.

  • Стиль линии, Стиль маркера, Стиль выделенного маркера и Стиль выбранного маркера, которые будут использоваться при отображении точек и линии маршрута.
