Справочник "Правила распределения косвенных расходов"

Для формирования правил распределения косвенных расходов по Статьям расходов используется одноименный справочник Правила распределения косвенных расходов. Распределение косвенных расходов осуществляется путем Закрытия месяца и служит для распределения накопленных расходов на Рейсы, Транспортные средства и Направления деятельности.


Для каждого Правила распределения косвенных расходов указываются Варианты распределения по приоритетности использования. Согласно установленному Варианту распределения распределяются расходы согласно установленным параметрам. Нераспределенные остатки расходов распределяются по следующему Варианту распределения согласно списку до момента окончания нераспределенных остатков расходов или списка Вариантов расходов.

Примечание

Использование Варианта распределения только «Без аналитики» может быть использовано при распределении общих расходов на заданный период.
