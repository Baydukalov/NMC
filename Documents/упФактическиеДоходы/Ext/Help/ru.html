Документ "Фактические доходы"

Документ Фактические доходы используется для ввода фактических доходов по Заявкам на перевозку груза. 


В документе указывается:

  • Заказчик, Соглашение с заказчиком, Контрагент заказчика.

  • Способ ввода доходов, возможно выбрать из:

    • Доходы по нескольким заявкам. При установке данного способа ввода доходов становится доступна закладка Список заявок для заполнения доходов, на которой указывается список Заявок на перевозку груза от конкретного Заказчика.

    • Доходы по одной заявке.

  • Заявка на перевозку груза. Документ основание, в котором рассчитаны плановые доходы.

  • На закладке Доходы указываются фактические доходы полученные от исполнения Заявки.

    • По команде Заполнить по заявке происходит заполнение строки документа данными по доходам из Заявки. В строке указывается Статья доходов, Валюта, Цена, Количество, Скидка, Сумма скидки, Сумма, Ставка НДС, Сумма НДС.

    • По команде Рассчитать/Рассчитать с расшифровкой происходит расчет доходов по статьям указанным в соглашении с Заказчиком перевозки.

    • По команде Перевыставить расходы по рейсу происходит заполнение строк документа данными по расходам из документа Фактические расходы по рейсу. С учетом правил распределения расходов по Заданиям на перевозку груза, которые созданы на основании Заявки на перевозку гру-за или являются ее частью (в случае исполнения Заявки по цепи поставок).

    • По команде Добавить указываются доходы полученные по заявке. Заполнение по данной команде возможно в случае отсутствия установленных тарифов с Организацией.

  • На закладке Дополнительно указывается Автор документа и Комментарий к документу.
