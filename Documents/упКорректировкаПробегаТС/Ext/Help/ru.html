Документ "Корректировка пробега ТС"

Отражение изменения данных по документу Рейс в системе возможно с помощью документа Корректировка рейсов.

При необходимости внесения изменений, отмены или добавления Заданий на перевозку груза в рейс, замены исполнителя по рейсу, создается документ Корректировка рейса, доступный по ссылке панели раздела Планирование рейсов. Документ Корректировка рейса доступен при подключении модуля Управление перевозками и Управление траспортом и учет ГСМ.

Документ Корректировка рейсов возможно создать:

  • На основании документа Рейс или из списка документов Корректировки рейсов вручную.

Основанием для создания документа Корректировка рейсов является документ Рейс.

Если документ Корректировка рейсов создается вручную, то необходимо указать рейс, по которому будут вноситься изменения. На основании документа Рейс заполняются реквизиты табличных частей Перевозчика, Заданий, Маршрута, Сотрудников, Параметров перевозки.

Изменения в документ Рейс с помощью документа Корректировка рейсов возможно вносить, когда рейс находится в статусе: К выполнению, Выполняется.

В случае изменения данных табличной части Перевозчика, необходимо указать какой партнер-перевозчик будет исполнять рейс, его контрагента, тип ТС, ТС и провести документ, тем самым в документе Рейс произойдет замена первоначальных данных.

На закладке Задания указываются все Задания на перевозку входящие в состав рейса.

  • По команде Добавить или по команде Подобрать возможно добавить задания на перевозку груза в документ Корректировка рейса.
  • По команде Распределить задания Происходит распределение заданий между заданиями, которые ещё не исполнены по рейсу.
  • По команде Отменить возможно отменить или перенести исполнение конкретного Задания на перевозку груза, путем подбора проблемы в точке.

На закладке Маршрут указывается порядок объезда точек по рейсу, факт их выполнения, а также задания, которые уже выполнены или необходимо выполнить в точке.

  • По команде Добавить возможно добавить адрес, который необходимо посетить водителю в ходе выполнения рейса. По команде Добавить табличной части Заданий возможно добавить задание по точке которое необходимо выполнить. По команде Переместить точку вверх или Переместить точку вниз возможно перемещение адресов согласно порядку объезда точек, при этом показатели Путь до точки (план), Время до точки (план), Прибытие (план) пересчитываются автоматически.
  • По команде Отменить возможно отменить точку маршрута.
  • Если отменить первую точку маршрута (погрузка), то происходит полная отмена рейса.
  • Если отменить точку разгрузки, когда рейс ещё не исполнялся, то автоматически исключиться задание на погрузку груза по данному заданию.
  • Если отменить точку разгрузки, когда первая точка (погрузка) уже пройдена, то сформируется дополнительное задание на возврат груза в точку отправления.
  • По команде Контрольная точка возможно добавить в документ рейс контрольную точку на маршруте, которую необходимо посетить в ходе исполнения рейса.

На закладке Сотрудники указываются сотрудники, исполняющие рейс. В случае замены водителя, второго водителя или экспедитора необходимо произвести замену путем выбора из выпадающего списка сотрудников.

На закладке Параметры перевозки указываются Зона погрузки и Зона разгрузки, которые в случае необходимости также возможно изменить.

На закладке Дополнительно указывается автор документа и комментарий.
