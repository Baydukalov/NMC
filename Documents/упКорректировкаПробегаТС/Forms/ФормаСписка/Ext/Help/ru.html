Документ "Корректировка пробега ТС"

Документ Корректировка пробега ТС используется для ввода текущего пробега ТС, а также различных корректировок пробегов ТС в конкурентном транспортном средстве.

Список документов Корректировка пробега ТС может быть открыт при помощи команды Все функции.

В документе указывается:

  • Транспортное средство для которого отражается факт корректировки текущего пробега.

  • Текущий пробег.
