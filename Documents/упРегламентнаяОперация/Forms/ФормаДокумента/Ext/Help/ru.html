Документ "Регламентная операция"

Документ Регламентная операция позволяет заносить операции с указанием статуса.

При выполнении процедуры закрытия месяца для каждой регламентной операции, входящей в состав процедуры в системе создается и проводится документ Регламентная операция с соответствующим видом регламентной операции. Список документов Регламентная операция может быть открыт при помощи одноименной команды панели навигации раздела Управление транспортом и учет ГСМ из группы Регламенты.
