Документ "Запрос условий перевозки"

Документ Запрос условий перевозки используется для первоначального запроса стоимости и сроков перевозки. Документ как правило оформляется менеджером в КИС с последующей выгрузкой в Axelot: Х4 TMS Управление транспортом и перевозками. В запросе могут быть указаны основные параметры груза. После чего в системе существует возможность рассчитать рамочную стоимость перевозки, сроки ее исполнения и т.д. На основании запроса условий перевозки может быть создана Заявка на перевозку груза. 


В случае выгрузки данных из КИС в Запросе условий перевозки автоматически заполняются реквизиты Номер КИС и Дата, Номер и Дата формирования документа-инициатора в корпоративной информационной системе и являющийся основанием для Запроса условий перевозки в TMS.

Перевозку грузов можно осуществлять собственным транспортом, либо привлеченным со стороны. Исходя из этого рассчитываются либо расходы, либо доходы на перевозку груза.

При создании нового Запроса условий перевозки для него автоматически устанавливается статус Новый. В этом статусе документ может быть отредактирован пользователем или отменен.

Изменение статуса документа происходит при выполнении конкретных событий в информационной базе.

При формировании Запроса также необходимо заполнить реквизиты Заказчик и Соглашение для того, чтобы в дальнейшем иметь возможность получать информацию по Запросам в разрезе клиентов, заказавших перевозку.

Для возможности расчета доходов или расходов по перевозке необходимо иметь информацию о точке погрузки и о точке разгрузки.

Информация о плановом маршруте перевозки заносится и отображается в табличной части Запрашиваемые параметры перевозки.

В Группе реквизитов Запрашиваемые параметры перевозки указывается Зона отправки, Зона получения, Адрес отправки груза и Адрес получения, Параметры груза, такие как масса, объем, количество мест, оценочная стоимость, температурный режим, класс опасности и другие, которые могут повлиять на стоимость перевозки.

По указанным параметрам Запроса можно произвести расчет плановых доходов перевозки для заказчика. Для этого нужно указать перечень услуг и суммы по каждому из них на закладке Плановые доходы.

Плановые доходы могут быть изначально заданы в договоре с Контрагентом Запроса условий перевозки. При использовании типовых доходов в текущем Запросе, необходимо воспользоваться командой Заполнить типовыми. В случае необходимости внесения дополнительных доходов, согласованных с заказчиком, но не вошедших в список типовых, необходимо добавить их в табличную часть, воспользовавшись командой Добавить.

Подбор доходов осуществляется из справочника Статьи доходов и расходов с типом статьи Статья доходов.

С учетом указанных параметров произойдет заполнение табличной части закладки Плановых доходов по Запросу условий перевозки.

По указанным параметрам Запроса можно произвести расчет плановых расходов перевозки для заказчика, где Заказчиком может выступать собственная организация.

Для этого нужно указать перечень расходов и суммы по каждому из них на закладке Плановые расходы.

Плановые расходы могут быть изначально заданы в договоре с Контрагентом Запроса условий перевозки. При использовании типовых расходов в текущем Запросе, необходимо воспользоваться командой Заполнить типовыми. В случае необходимости внесения дополнительных расходов, согласованных с заказчиком, но не вошедших в список типовых, необходимо добавить их в табличную часть, воспользовавшись командой Добавить.

Для внесения в Запрос расходов, согласованных с заказчиком, необходимо добавить их в табличную часть, воспользовавшись командой Добавить.

Подбор расходов осуществляется из справочника Статьи доходов и расходов с типом статьи Статья расходов.

С учетом указанных параметров произойдет заполнение табличной части закладки Плановых расходов по Запросу условий перевозки.

Вместе со стоимостью перевозки возможно указать Срок доставки и Дату возможной доставки груза.

Для всего документа на закладке Дополнительно заполняется информация об Авторе документа – Пользователе, создавшем документ в случае его ручного ввода, Менеджере, Подразделении, Логисте, а также сводная информация по Запросу условий перевозки в поле Комментарий.
