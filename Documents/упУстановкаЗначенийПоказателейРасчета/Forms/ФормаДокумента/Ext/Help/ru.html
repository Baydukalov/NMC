Документ "Установка значений фиксированных показателей"

При помощи документа Установка значений фиксированных показателей в системе фиксируются значения тех показателей расчета тарифов, для которых установлен тип показателя Фиксированный. 

Значение фиксированного показателя задается для организации. Значение фиксированного показателя со временем может меняться. Дата начала действия значения фиксированного показателя указывается в реквизите Дата.

Перечень фиксированных показателей и их значения указываются в табличной части документах.
