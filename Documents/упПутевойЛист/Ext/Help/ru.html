Документ "Путевой лист"

Путевой лист – основной первичный документ учета работы водителя и пробега транспортного средства, маршрута, выда-ваемый ежедневно (при необходимости) водителям транспортных средств. Как правило, Путевой лист выписывается только на один день (смену) работы водителя. Следующий Путевой лист выдается при условии, что водитель сдал документ за предыдущую смену. 

Выдача Путевого листа осуществляется на основании документа Рейс.

Для работы с Путевыми листами необходимо:

  • Установить признак Используется учет путевых листов в Настройках системы (раздел Администрирование – Настройки системы – Учет ГСМ и управление ТС).

  • В элементе справочника Партнер на закладке Документация установить признак Требуется оформление путевого листа.

  • В Виде перевозки на закладке Настройки рейса установить признак Требуется оформление путевого листа.

Путевой лист возможно создать:

  • По команде Создать на основании формы документа Рейс.

  • С помощью РМ: Водителя.

  • С помощью РМ: Диспетчера гаража.

При создании Путевого листа реквизиты документа частично заполняются данными, заполненными в элементе справочника Транспортное средство, на которое выдается Путевой лист.

На закладке Главное путевого листа указывается:

  • Начальный пробег по одометру, используется в случае сброса данных и отсчета пробега заново в ТС.

  • Пробег начальный. Указываются данные закладки Пробег/ГСМ справочника транспортного средства.

  • Температура воздуха.

  • Транспортное средство. Указывается транспортное средство, которое назначено для выполнения рейса.

  • Масса прицепного состава. Учитывается при расчете нормативного расхода ГСМ.

  • Гараж. Указываются данные закладки Гаражи справочника транспортного средства, место начала рейса.

  • Рейс маршрута.

  • Включить рейс в путевой лист. Данная опция включается в Настройках системы (раздел Администрирование – Настройки системы – Учет ГСМ и управление ТС – Разрешить выписку одного путевого листа на несколько рейсов). По данной команде в один путевой лист возможно включить несколько рейсов. Включение рейсов в один путевой лист необходимо в случае, когда новый путевой лист выписать на одно и тоже транспортное средство невозможно.

  • Начало рейса. Указывается дата и время начала рейса.

На закладке Транспортное средство указывается:

■        Начальный остаток ГСМ. Указываются данные закладки Пробег/ГСМ справочника транспортного средства.

Плановые показатели расхода ГСМ с учетом общей длины маршрута по рейсу задаются на закладке Транспортное средство/Расход ГСМ Путевого листа. Данные показатели заполняются на основании норм расхода ГСМ, установленных в Модели ТС или в документе Установка норм расхода по каждому отдельному ТС.

По команде Добавить табличной части Расход ГСМ осуществляется ввод дополнительных расходных ГСМ. 

По команде Заполнить ГСМ производится заполнение табличной части закладки Расход ГСМ, в случае если нормы расхода не заданы для транспортного средства путевого листа. Команда Заполнить     ГСМ   не активна при установленных нормах расхода ГСМ у ТС.

      По команде Рассчитать Плановый расход топлива осуществляется заполнение плановых расходов ГСМ в зависимости от плановой длины маршрута, на основании документа рейс.

На закладке Доп. оборудование указывается:

  • Общее. Дополнительное оборудование, по которому необходимо вести учет расхода ГСМ.

  • Общее. Транспортное средство, при учете того, что доп. оборудование установлено на ТС, а не использует собственный бак доп. оборудования.

  • Общее. Начальные остатки по наработанным моточасам.

  • Общее. Табличная часть Режимов работы, в которой можно задать моточасы по каждому режиму работы.

На закладке Доп. оборудование/Расход ГСМ указываются: Вид ГСМ, Доп. оборудование, признак использования Расхода из собственного бака, Расход топлива с ТС, Норма расхода, Начальный остаток, Конечный остаток, рассчитывается Плановый расход, Расход по норме, Фактический расход и др.

При завершении путевого листа заполняются следующие данные:

  • Окончание рейса. Указывается дата и время окончания рейса.

  • Пробег конечный. Указываются фактические показатели датчиков транспортного средства, отвечающие за пробег. При наличии установленных датчиков у ТС возможно заполнение по команде Рассчитать по данным датчиков.

  • Конечный остаток ГСМ. Указываются фактические показатели датчиков транспортного средства, отвечающие за учет ГСМ. При наличии установленных датчиков у ТС возможно заполнение по команде Рассчитать Конечный остаток по данным датчиков.

  • Конечные моточасы. Указываются фактические показатели датчиков доп. оборудования, отвечающие за наработанные моточасы.

  • Конечный остаток ГСМ в собственном баке доп. оборудования. Указываются фактические показатели датчиков транспортного средства, отвечающие за учет ГСМ.

  • По команде Рассчитать Нормативный расход топлива осуществляется заполнение Расхода топлива по норме на основании пробега, введенного по итогам исполнения рейса.
  • По команде Рассчитать Конечный остаток по нормативному расходу осуществляется заполнение Конечного остатка топлива с учетом нормативного расхода.
  • По команде рассчитать Конечный остаток по данным датчиков осуществляется заполнение Конечного остатка топлива по данным датчика. 

На закладке Заправки/сливы по команде Добавить, заполняются данные о заправке или сливе ГСМ, где указываются:

  • Вид ГСМ, Горюче-смазочные материалы.

  • Вид движения. Указывается либо заправка, либо слив топлива.

  • Дата. Указывается дата заправки/слива ГСМ.

  • Документ расхода, Номер, Дата, Наименование документа.

  • Валюта. Указывается валюта, в которой производилась заправка.

  • Цена ГСМ.

  • Количество ГСМ, которым было заправлено/слито транспортное средство.

  • Стоимостные показатели: Ставка НДС, Стоимость, Сумма НДС.

  • Если производится слив ГСМ, то необходимо указать Приемник слива:

    • Транспортное средство;

    • Склад.

  • Если заправка производится с использованием топливной карты, то возможно указать:

    • АЗС. Указывается партнер-АЗС.

    • Топливная карта.

На закладке Сотрудники указываются: Механик, который производил осмотр ТС и Врач, который производил медицинский осмотр во-дителя. Ф.И.О. указанных сотрудников указывается в печатной форме Путевого листа.

По итогам заполнения Путевого лист его статус изменяется на Закрыт.

В процессе работы Путевой лист может находиться в различных состояниях, которые определяются установленным статусом документа:

  • Новый. Документ Путевой лист создан и записан.

  • Выдан. Документ Путевой лист выдан водителю транспортного средства для исполнения рейса.

  • Возвращен. Документ Путевой лист возвращен водителем транспортного средства.

  • Закрыт. Документ Путевой лист закрыт.
