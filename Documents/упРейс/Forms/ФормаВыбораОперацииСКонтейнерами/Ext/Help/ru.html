Документ Операции с контейнерами

Документ Операции с контейнерами предназначен как для простого учета формирования и расформирования контейнера, так и для сложного учета движения контейнера и установки контейнера на Транспортные средства.

Доступны следующие варианты создания документа Операции с контейнерами.
  • Вручную Пользователем
    • Из списка документов Операции с контейнерами. В группе Формирование контейнеров раздела Планирование необходимо выбрать список Операций с контейнерами. По команде Создать открывается новый документ Операции с контейнерами. В шапке документа необходимо выбрать Вид перевозки, определить время формирования. В табличной части закладки Заданий необходимо подобрать Задания на перевозку груза, по которым будет исполняться Операция с контейнером. Подбор Заданий на перевозку груза осуществляется с учетом заданного Вида перевозки. На основании подобранных заданий заполняются значения массы, объема и прочие показатели груза.

    • Путем ввода на основании Задания на перевозку груза. По команде Создать на основании Операции с контейнером открывается документ Операции с контейнером, где происходит заполнение Вида перевозки значением из Задания на перевозку груза, указываются параметры массы, объема и т.д. В сформированные на основании задания Операции с контейнерами возможно подобрать задания с совпадающим Видом перевозки.

    • В РМ: Формирование контейнеров, где производится объединение Заданий на перевозку груза по заданным парам адресов.

Для документа Операции с контейнерами заполнение данных производится аналогично типовому заполнению документа Рейс.

Для каждого документа заполняется следующая информация:
  • Владелец – Владелец контейнера
  • Контрагент – юридическое лицо владельца контейнера
  • Соглашение – Соглашение с перевозчиком, используемое в рамках Операции с контейнерами
  • Тип контейнера
  • Контейнер

В системе можно отслеживать факт стаффировки и расформирования контейнера. Для этого предусмотрен отдельный документ Выполнение операции, который может быть представлен в виде документов:
  • Формирование контейнера
  • Расформирование контейнера
Документ Выполнение операции возможно создать на основании документа Операции с контейнером.

В документе Выполнение операции необходимо указать дату и время Прибытия в точку, Начала работ, Убытия из точки. На основании документа Операции с контейнером заполняется реквизит Адрес точки, указанный к выбору в порядке объезда точек по маршруту по данной Операции с контейнером. На основании выбранного Адреса точки происходит заполнение табличной части документа Выполнение операции, где указывается Задание на перевозку груза, Операция в точке, Грузовые места.

Необходимо отметить факт выполнения задания по каждой конкретной точке маршрута и провести документ.

При фиксации проблемы в точке производятся корректирующие действия с Заданием на перевозку, по которому зафиксирована проблема. Возможны следующие действия:

  • Вернуть Задания на перевозку груза обратно в планирование.
    • Если Задание на перевозку груза не выполнено целиком, необходимо в поле Действие документа Проблемы в точке указать Вернуть в планирование. В этом случае статус Задания на перевозку груза устанавливается в значение «К планированию». Задание на перевозку или грузовое место возможно спланировать в Операцию с контейнером ещё раз.

      Важно! Вернуть в планирование частично не выпол-ненное Задание на перевозку груза возможно, если в Виде перевозки значение реквизита Информация о грузовых местах установлена как Вводится в заявке на перевозку.
  • Отменить Задание на перевозку груза.
    • Если Задание на перевозку груза не выполнено целиком, в поле Действие документа Проблемы в точке необходимо указать Отменить задание. В этом случае статус Задания на перевозку груза устанавливается в значение «Отменено».
      Важно! Отменить частично не выполненное Задание на перевозку груза возможно, если в Виде пе-ревозки значение реквизита Информация о грузовых местах установлена как Вводится в заявке на перевозку.
