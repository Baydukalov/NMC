Документ "Фактические расходы по рейсу"

В документе Фактические расходы по рейсу указывается: 

  • Партнер – указывается Партнер-перевозчик 

  • Соглашение – указывается соглашение с Партнером-перевозчиком 

  • Контрагент – указывается контрагент Партнера-перевозчика 

  • Способ ввода расходов – указывается один из способов ввода расходов: 

    • Расходы по одному рейсу – расходы вводятся на основании одного рейса. При данном способе ввода расходов доступно поле Рейс, в котором указывается Рейс, для которого будут указаны расходы. 

      На закладке Расходы указываются данные: 

      • Задание – указывается Задание, на которое распределяется статья расходов. 

      • Статья расходов – указывается статья расходов по рейсу. 

      • Правило распределения – указывается принцип распределения расходов. 

      • Валюта – валюта расходов 

      • Количество – количество указанных расходов 

      • Цена – цена расходов 

      • Сумма – сумма расходов 

      • Ставка НДС, Сумма НДС – указываются ставка и сумма НДС 

      По команде Заполнить по рейсам список расходов заполняется согласно плановым расходом рейса. По команде Рассчитать происходит расчет расходов согласно Соглашению с партнером. 

    • Расходы по нескольким рейсам – расходы вводятся конкретно на каждый заданный рейс. При данном способе ввода расходов на закладке Расходыдля каждой статьи расходов указывается Рейс, к которому данные расходы относятся. Команда Заполнить по рейсам для данного ввода расхода недоступна. 

    • Обобщенные расходы по нескольким рейсам – расходы распределяются по рейсам согласно заданным правилам распределения. При данном способе ввода расходов становится доступна закладка Список рейсов для распределения расходов. На данной вкладке задаются Рейсы для распределения расходов, заданных на закладке Расходы, согласно правилам распределения расходов.
