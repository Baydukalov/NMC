Документ "Корректировка рейсов"

Отражение изменения данных по документу Рейс в системе возможно с помощью документа Корректировка рейсов.

При необходимости внесения изменений, отмены или добавления Заданий на перевозку груза в рейс, замены исполнителя по рейсу, создается документ Корректировка рейса.

Документ Корректировка рейсов возможно создать:

  • На основании документа Рейс или из списка документов Корректировки рейсов вручную.

Основанием для создания документа Корректировка рейсов является документ Рейс.

Если документ Корректировка рейсов создается вручную, то необходимо указать рейс, по которому будут вноситься изменения. На основании документа Рейс заполняются реквизиты табличных частей Перевозчика, Заданий, Маршрута, Сотрудников, Параметров перевозки.

Изменения в документ Рейс с помощью документа Корректировка рейсов возможно вносить, когда рейс находится в статусе: Назначено ТС, К выполнению, Выполняется, Выполнен частично, Выполнен, Завершен.

В случае изменения данных табличной части Перевозчика, необходимо указать какой партнер-перевозчик будет исполнять рейс, его контрагента, тип ТС, ТС, первого/второго водителя, экспедитора и провести документ, тем самым в документе Рейс произойдет замена первоначальных данных.

На закладке Задания указываются все Задания на перевозку входящие в состав рейса.

  • По команде Добавить или по команде Подбор возможно добавить задания на перевозку груза в документ Корректировка рейса.

  • По команде Распределить происходит распределение заданий между заданиями по рейсу.

  • По команде Отменить/Вернуть возможно отменить или перенести исполнение конкретного Задания на перевозку груза, путем подбора проблемы в точке.

На закладке Маршрут указывается порядок объезда точек по рейсу, факт их выполнения, а также задания которые уже выполнены или необходимо выполнить в точке.

  • По команде Добавить возможно добавить адрес, который необходимо посетить водителю в ходе выполнения рейса. По команде Добавить табличной части Заданий возможно добавить задание по точке которое необходимо выполнить. По команде Переместить точку вверх или Переместить точку вниз возможно перемещение адресов согласно порядку объезда точек, при этом показатели Путь до точки (план), Время до точки (план), Прибытие (план) пересчитываются автоматически.

  • По команде Выполнить возможно подтвердить факт выполнения точки маршрута.

  • По команде Отменить возможно отменить точку выезда/возврата в гараж.

  • По команде Контрольная точка возможно добавить в документ рейс контрольную точку на маршруте, которую необходимо посетить в ходе исполнения рейса.

  • По команде Разделить работы возможно деление не-скольких типов операций в точке, с возможностью установки новой точки.

    На закладке Дополнительно указываются Зона погрузки и Зона разгрузки, которые в случае необходимости также возможно изменить. А также указывается Автор документа, Источник создания, автор изменения, Часовой пояс, Комментарий к документу Корректировка рейса.
