Документ "Прочие расходы"

Для регистрации расходов, которые не связаны непосредственно с оказанием услуг перевозки, т.е. косвенных расходов в системе используется документ Прочие расходы. К таким расходам можно отнести услуги сторонних контрагентов, которые могут быть связаны с обслуживанием парка транспортных средств, гаражей, обеспечением дополнительного сервиса при оказании услуг перевозки и т.д.
Например!

Мойка парка транспортных средств, предоставление услуг GSM-мониторинга и т.д.

В документе указывается:

  • Организация – получатель услуг.

    • Если Организация не указана, значит в системе заведена только одна, учитывается по умолчанию.

  • Партнер, оказывающий услуги, Соглашение, в соответствии с условиями которого оказываются услуги и Контрагент, который будет выгружаться из системы для ведения бухгалтерского учета.

  • Период распределения затрат, который определяет на рейсы какого периода будут распределяться указанные затраты.

  • Вид прочих расходов, который определяет правило определения списка рейсов, на которые будут распределятся указанные расходы.

    • Если указан вид По транспортному средству, то расходы будут распределяться на рейсы, выполненные выбранным в документе транспортным средством.

    • При выборе вида По водителю/ экспедитору, расходы будут распределяться на рейсы, выполненные выбранным в документе водителем/экспедитором.

    • Непосредственно список рейсов, на которые необходимо распределять расходы, указывается в табличной части на закладке Список рейсов для распределения, которая становится доступной при выборе вида прочих расходов По рейсам.

    • При выборе вида прочих расходов По виду перевозки расходы будут распределятся по всем рейсам указанного вида перевозки.

    • Вид прочих расходов Общие за период позволяет распределить расходы на все рейсы попавшие в период распределения.

На закладке Расходы для каждой полученной услуги указываются статьи расходов, которые могут быть использованы для отражения расходов в бухгалтерском учете, правило распределения по рейсам из справочника Правила распределения расходов, количество, валюта, сумма в валюте, ставка НДС и НДС.
