Документ "Входящая претензия"

Документ Входящая претензия используется для фиксации претензии от партнеров и распределения расходов, по итогам ее удовлетворения.

Документ Входящая претензия может быть создан на основании Заявки на перевозку груза, Рейса или путем создания вручную из списка документов Входящие претензии. 

В документе указывается:

  • Организация, Партнер, Контрагент партнера, Соглашение.

  • Причина претензии. Используется для классификации претензий.

  • Документ основание. Указывается документ, на основании которого регистрируется Входящая претензия.

  • На закладке Описание указываются:
    • Заявка на перевозку груза;

    • Рейс. На указанный рейс, в случае удовлетворения претензии, распределяются расходы;

    • Описание претензии. Детальное описание входящей претензии.

    Важно!

    Для распределения расходов указывать Рейс обязательно.

  • На закладке Результат указывается результат обработки и стоимостные показатели по расходу по Входящей претензии.

    Важно!

    В случае, если не указать конкретное Задание на перевозку груза в строке расходов, то расходы распределятся по всем Заданиям на перевозку груза в рейсе согласно Правилу распределения.

  • На закладке Участники указываются Партнеры и их Контактные лица, участвующие в зарегистрированной претензии.

  • На закладке Дополнительно указывается Автор документа и Комментарий к документу.

Документ Входящая претензия может принимать следующие статусы:

  • Новая. Документ Входящая претензия создан и записан.

  • В работе. Документ Входящая претензия взят в работу.

  • Удовлетворена. По документу Входящая претензия все претензии Партнера удовлетворены..

  • Удовлетворена частично. По документу Входящая претензия претензии Партнера удовлетворены не в полном объеме..

  • Отклонена. По документу Входящая претензия все претензии Партнера не удовлетворены.

  • Отменена. Документ Входящая претензия отменен пользователем.
