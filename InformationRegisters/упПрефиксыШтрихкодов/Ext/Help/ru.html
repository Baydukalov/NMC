Регистр сведений Префиксы штрихкодов

Для хранения данных о префиксах штрихкодов объектов системы предусмотрен регистр сведений "Префиксы штрихкодов".

Например,

задание префиксов применимо для печатных форм складских документов для точного определения типа операции при сканировании. Регистр сведений Префиксы штрихкодов представляет одноуровневую структуру и содержит данные о префиксе объекта и ссылки на объект, заполняемой в поле Тип объекта.
