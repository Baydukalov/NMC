Отчет "Анализ расходов по рейсам"

Отчет Анализ расходов по рейсам позволяет выполнять план-фактный анализ по расходам рейса в разрезе статей затрат и заданий на перевозку груза. В отчете отображаются параметры рейса с детализаций по заданиям. Для удобства работы с отчетом существует возможность установки отбора по параметрам рейса, например, по перевозчику, или по виду перевозки и др. Обязательным отбором является Период формирования отчета.

Аналитические разрезы используются для детализации расходов с детализацией до грузовых мест. При задании отдельных аналитических разрезов возможно распределить расходы по группам товаров с дальнейшим анализом затрат в отчете Анализ расходов по рейсам.
