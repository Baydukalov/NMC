Рабочее место "Закрытие месяца"

Рабочее место Закрытие месяца используется для выполнения регламентных операций с целью регламентированного учета.

Рабочее место Закрытие месяца может быть открыто при помощи одноименной команды панели навигации раздела Управление транспортом и учет ГСМ из группы Регламенты.

Закрытие месяца выполняется помесячно за период, указанный в одноименном реквизите. Запуск выполнения процедуры закрытия месяца осуществляется при помощи команды Закрыть период. Успешно выполненные операции в рабочем месте выделяются зеленым цветом шрифта, выполненные с ошибками - красным, а невыполненные или пропущенные - черным. Общую информацию о состоянии выполнения регламентных операций можно также проконтролировать в строке состояния, которая выводится в конце списка всех операций. Для отмены выполнения регламентных операций за указанный в рабочем месте месяц используется кнопки Отменить закрытие.

При выполнении процедуры закрытия месяца для каждой регламентной операции, входящей в состав процедуры в системе создается и проводится документ Регламентная операция с соответствующим видом регламентной операции. Список документов Регламентная операция может быть открыт при помощи одноименной команды панели навигации раздела Управление транспортом и учет ГСМ из группы Регламенты.
