Обработка "Генерация топологии склада"

Для автоматического формирования больших массивов ячеек применяется обработка "Генерация топологии склада".

Обработка "Генерация топологии склада" доступна на панели раздела Настройка склада и операций.

Заполнение формы обработки "Генерация склада" для создания новых ячеек начинается с задания Зоны склада, для которой ячейки необходимо сформировать. Значение Зоны выбирается из списка одноименного справочника Зоны склада.

Настройка правил формирования адреса, а также интервалы номеров ячеек, которые необходимо создать, осуществляется на вкладке Адресация.

Адрес ячейки может состоять из следующих параметров:

■      Проезд;

■      Стеллаж;

■      Ярус;

■      Позиция.

В форме обработки необходимо выбрать только те составляющие адреса ячейки, которые необходимо использовать при генерации топологии.

Для каждой составляющей адреса необходимо заполнить следующие параметры:

■      Тип номера – значение, определяющие диапазон допустимых символов, из которых будет формироваться адрес ячейки. Тип номера может быть Числовой (например, 1,2,3) или Алфавитно-Цифровой (лат.) (например, A,B,С или A1, А2, B1, B2).

■      Для ячейки может быть задана фиксированная длина составляющей кода ячейки. Например, при заданной длине для проезда «2», значение кода первого проезда будет равно «01».

Важно!

Задание количества знаков фиксированной длины позволит правильно сортировать ячейки при десятичных числовых значениях, то есть правильно отображать числовой ряд ячеек с 1 по 10. При этом к числу добавляется нужное количество лидирующих нулей, и нумерация ячеек будет выглядеть следующим образом: 01, 02, 03, 04, 05 и т.д.


■      Номера от… Номера до – диапазон номеров ячеек, которые необходимо создать.

■      Разделить – Символ, который будет служить разделителем между значениями проезда, стеллажа, яруса и позиции при формировании кода ячейки.

Например, для ячейки с кодом 1-1-2-3-1 разделителем будет являться символ «-(дефис)».

Для кода ячейки также может быть заданы префикс - выражение, с которого будет начинаться код ячейки на текущем складе, и суффикс – выражение, которое будет завершать код ячейки.

При установленном признаке Обновлять существующие при формировании новой структуры склада производится перезапись ранее созданных ячеек, входящих в заданный диапазон формирования.

Для создания структуры склада согласно введенным правилам формирования адресов ячеек предусмотрена кнопка Создать структуру. По нажатию кнопки в справочнике Ячейки будет сформирован набор адресов по заданным правилам согласно заданному диапазону.

Важно!

При формировании кодов ячеек выполняется проверка уникальности кодов и штрихкодов ячеек. В случае неуникальности значений в окне сообщений выводится соответствующее предупреждение с указанием неуникального объекта.
