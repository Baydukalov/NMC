"РМ: Планирование ремонтов"

Рабочее место РМ: Планирование ремонтов предназначено для планирования ремонтов.

Интерфейс рабочего места включает слева область для визуального представления графика проведения ремонтных работы и справа панель команд.

Данные в рабочем месте отображаются для выбранного гаража с детализацией по ямам. Длительность периода отображения данных в графике выбирается при помощи кнопки Период отображения и может принимать одно из следующих значений: День, Неделя, Месяц. Непосредственно интервал дат, за которые отображаются данные в графике устанавливается по текущей дате. Изменить интервал просмотра можно при помощи кнопок и . Кроме этого, интервал просмотра можно изменить по дате, входящей в период отображения при помощи кнопки .

Документы Ремонт (ТО) отображаются в графике в зависимости от статуса документа. Период выполнения ремонтных работ по документу в статусе Запланирован отображается желтым фоном, в статусе Начат – зеленым, а в статусе Завершен – розовым.

При выделении области периода выполнения ремонтных работ по конкретному документу в панели команд появляется команды, при помощи которых можно открыть выделенный документ, транспортное средство, для которого запланировано выполнение ремонтных работ или вид ремонта.

В панели команд при помощи группы реквизитов Новые ремонты можно создать новый документ Ремонт (ТО). Кроме этого новый документ Ремонт (ТО) можно создать в области для визуального представления графика проведения ремонтных работы выделив планируемый период выполнение при помощи левой кнопки мыши.
