Обработка "Загрузка данных от топливных компаний"

Загрузка данных от топливных компаний предназначена для возможности автоматического ввода данных в систему, полученных от топливных компаний.

Загружаемые данные сохраняются в системе и позволяют в дальнейшем сопоставить данные заправок путевых листов и заправок топливных компаний в рамках одного отчета – Анализ данных о заправках от топливных компаний.

Для выполнения загрузки данных необходимо:

  • Открыть обработку Загрузка данных от топливных компаний.

  • По команде Выбрать файл производится подбор необходимого файла. Как правило, файлы от топливных компаний хранятся и предоставляются в формате«.xml» один раз в неделю или более. При выборе файла происходит заполнение табличной части документа.

  • По команде Сопоставить загруженные данные распределяются по данным о заправках по следующим правилам:

    • Период записи из файла попадает в период работы по путевому листу, совпадают топливные карты, вид ГСМ и количество топлива отличается не более чем на 1л;

    • Период записи из файла попадает в период работы по путевому листу, совпадают топливные карты, вид ГСМ;

    • Период записи из файла попадает в период работы по путевому листу, совпадают транспортные средства (определенное по топливной карте) и вид ГСМ.

  • По команде Записать данные будут записаны в регистр сведений Данные от топливных компаний.

Загруженные данные могут быть использованы для сравнительного анализа в отчете Анализ данных о заправках от топливных компаний. В отчете фиксируются данные по путевым листам, данные топливных компаний, заправки по путевым листам, данные по заправкам от топливных компаний.
