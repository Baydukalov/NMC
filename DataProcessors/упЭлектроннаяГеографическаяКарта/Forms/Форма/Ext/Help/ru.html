Электронная географическая карта

В конфигурации управление перевозками (TMS), использует географические объекты, адрес, зона (географическая зона), маршрут (путь между двумя географическими адресами).

Объекты отображаются в области расположенной с правой стороны формы обработки. С противоположной стороны, приведен список отображаемых объектов, разделенных на группы: адреса, зоны, маршруты.

Колонка "Объект" содержит наименование текущего объекта, колонка "Редактирование", дает возможность пользователю, изменять положение объекта, у поля колонки есть три значения: Редактировать, Завершить редактирование, Указать точку, область, маршрут.

Редактировать объект можно только один, также при редактировании гео - зоны, можно использовать реперные точки, вызвав командой контекстного меню "Отобразить реперные точки". Команда доступны только в режиме редактирования области зоны.

Команда добавить расположенная на командной панели, вызывает форму выбора для выбранной группы объектов.

Команда подбор, для группы адреса или зоны, вызывает формы подбора объектов
