РМ "Диспетчер гаража"

Рабочее место предназначено для работы с путевыми листами. Рабочее место доступно только при подключении модуля Управление транспортом.

Рабочее место Диспетчер гаража доступно по одноименной команде раздела Управление транспортом и учет ГСМ.

Рабочее место представляет собой список рейсов. В списке перечислена информация по рейсам и транспортным средствам.

По команде Выдать происходит создание документа Путевой лист на выделенный Рейс.

По команде Открыть происходит открытие документа Путевой лист выделенной строки. Данная команда доступна, если на рейс создан документ Путевой лист.

По команде Вернуть для выделенной строки документ Путевой лист изменяет статус на Возращен. При этом в документе должны быть заполнены поля Окончание, Конечный пробег, Конечный остаток, Расход по норме. Данная команда доступна для документов Путевой лист со статусом Выдан.

По команде Закрыть путевой лист для выделенной строки у документа Путевой лист изменяется статус на Закрыт. Данная команда доступна, если документ Путевой лист имеет статус Возвращен. Документ Путевой лист со статусом Закрыт недоступен для редактирования.
